# Prisma Wallpaper Manager

Prisma Wallpaper Manager is a **Windows desktop application** built with **Flutter**, designed to help you manage your wallpapers easily from one central place.

Instead of worrying about where wallpapers are stored, scattered folders, or file organization, Prisma gives you a clean and simple way to browse, manage, and apply wallpapers without any hassle.
